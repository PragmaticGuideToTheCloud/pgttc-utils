# Software install

## brew

To install terraform, kubectl and gcloud run:

    brew install terraform
    brew install kubernetes-cli
    brew install --cask google-cloud-sdk

## makefile

To install terraform, kubectl and gcloud run:

    make terraform
    make kubectl
    make cloudsdk

## versions

    terraform version   # 1.1.7
    kubectl version     # 1.21.8
